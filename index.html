Es una aplicaciòn multiplataforma (iOs y Android) desarrollada con React Native (mi primer proyecto) con el que tuve la oportunidad de aprender a desarrollar con esta herramienta, y Expo.

PLATZI FINANZAS

Es una aplicaciòn nativa para iOs, desarrollada con fines didàcticos para el curso de desarrollo de aplicaciones iOs de Platzi.
Se supone que es una herramienta donde puedes reflejar tus ingresos/egresos diarios, y tener un balance general.
La app es bastante sencilla, el fin primordial fue aprender a diseñar interfaces, preparar el entorno para el proyecto, hacer login con Email con redes sociales, consumir apis de forma nativa, programar "localNotifications" y "remoteNotifications" asi como tambien entender el universo de desarrollo para iOs.

JOCKEY

Se encuentra aun en fase de desarrollo.
Todos los componentes de UI/UX fueron diseñados por mì con el uso de la herramienta de Adobe Xd. Luego pase el diseño a la implementacion programatica con React Native. La arquitectura de la aplicaciòn esta basada en MVVM, implementando componentes agnosticos y reusables, totalmente personalizados. Utilizo como filosofìa de desarrollo TDD (Lo aprendì a las malas xD), las pruebas unitarias estan apoyadas con Jest, y la prueba e2e con Detox. El codigo esta escrito en typescript, usando eslint y las reglas de airbnb. Consume un web api rest json desarrollada en Laravel 6, y usa una BD desplegada en postgreSQL.

El app esta pensada para que los usuarios puedan apostar en 8 modos diferentes en diversos Hipodromos de cualquier paìs. El reto de implementaciòn en este proyecto realmente es el tema de la gestion de pagos.
